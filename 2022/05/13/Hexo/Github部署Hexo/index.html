Github部署Hexo

发表于 2022-05-13 更新于 2022-05-20 分类于 Hexo
Github部署Hexo

1、安装

下载并安装Node.js和Git 一直next即可

2、检测

检测Node.js是否安装成功，在命令行中输入 node -v
检测npm是否安装成功，在命令行中输入npm -v

3、注册

·注册github并建立新的仓库 + New repository 名字为用户名.github.io
·选择Public 和  Add a README file

4、公钥

·在Git Bash中，设置user.name和user.email配置信息：
·git config --global user.name "你的GitHub用户名"
·git config --global user.email "你的GitHub注册邮箱"
·生成ssh密钥文件：
·ssh-keygen -t rsa -C "你的GitHub注册邮箱"
·然后直接三个回车即可，默认不需要设置密码
·然后找到生成的.ssh的文件夹中的id_rsa.pub密钥，将内容全部复制到GitHub_Settings_keys 页面，新建new SSH Key
·在Git Bash中检测GitHub公钥设置是否成功，输入 ssh git@github.com

5、安装hexo

·使用npm命令安装Hexo，输入：npm install -g hexo-cli 这个安装时间较长耐心等待，安装完成后。
·初始化我们的博客，输入：hexo init blog 注意：这里的命令都是作用在刚刚创建的Blog文件夹中。

6、部署

部署工具 npm install hexo-deployer-git -save
deploy:
      type: git
      repo: https://github.com/wapchief/wapchief.github.io.git(https报错则换SSH)
     branch: master/main
hexo g -d
